## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Set up local database

```bash
$ docker-compose up -d
$ yarn migration:run
```

## Running the app

```bash
# development
$ yarn start

# production mode
$ NODE_ENV=production yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
